# How We Act Here

## Code of Conduct

We are committed to fostering a respectful, inclusive, and collaborative environment.

- Treat all contributors with respect and professionalism.

- Value ideas over ego and collaboration over competition.

- Communicate clearly, constructively, and with patience.

- Harassment, discrimination, or disruptive behavior is NOT tolerated.

This project thrives on shared effort and mutual respect.
